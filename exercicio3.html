Exercício 3 - Soma dos Elementos Aleatórios

Este exercício envolve a criação de uma matriz 7x7 com números aleatórios e a soma de todos os seus elementos.

Código do Exercício

function gerarMatrizAleatoria() {
    let matriz = [];
    for (let i = 0; i < 7; i++) {
        matriz[i] = [];
        for (let j = 0; j < 7; j++) {
            matriz[i][j] = Math.floor(Math.random() * 20) + 1; // Números aleatórios de 1 a 20
        }
    }
    return matriz;
}

function somaMatriz(matriz) {
    let soma = 0;
    for (let i = 0; i < matriz.length; i++) {
        for (let j = 0; j < matriz[i].length; j++) {
            soma += matriz[i][j];
        }
    }
    return soma;
}

function executarExercicio() {
    let matriz = gerarMatrizAleatoria();
    console.table(matriz);  // Exibe a matriz no console
    console.log("Soma dos elementos da matriz:", somaMatriz(matriz));  // Exibe a soma dos elementos
}

Explicação:

O código funciona da seguinte forma:

  • A função gerarMatrizAleatoria cria uma matriz de 7x7 com números aleatórios entre 1 e 20.
  • Dentro de dois laços for, percorre-se todas as linhas e colunas da matriz e, para cada posição, é gerado um número aleatório.
  • A função somaMatriz recebe a matriz gerada como argumento.
  • Dentro de dois laços for, percorre-se todos os elementos da matriz, somando-os à variável soma.
  • Após percorrer todos os elementos da matriz, a função retorna o valor total da soma.
  • A função executarExercicio chama as funções gerarMatrizAleatoria e somaMatriz, exibe a matriz no console e mostra a soma dos elementos.

Saída Esperada:

O resultado será a soma de todos os elementos da matriz 7x7 gerada aleatoriamente. O valor da soma pode variar a cada execução, pois os números da matriz são aleatórios.
